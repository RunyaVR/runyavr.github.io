La Modification de jeu

Doki Doki Literature Club! Est un jeu codé dans le langage ren'py, un langage dévié du Python qui est très simple à apprendre et ainsi les joueurs les plus dévoués peuvent apprendre à faire leurs propres histoires en reprenant les mécaniques de DDLC. Ils auront cependant besoin de quelques logiciels afin de faire leur tout premier mod:

  • ren'py 6.99.12
  • python 7
  • DDLC Mod Template 2.0
  • une copie de DDLC

Des prérequis comme le ren'py 6.99.12 and python 7 doivent être installés au préalable puisqu'ils sont requis pour modifier les fichiers du DDLC Mod Template. Le DDLC Mod Template est un incontournable puisqu'il contient une version du jeu vierge prête à être modifiée.
